# EliteHomes

EliteHomes is a polished static real estate landing page built with HTML, CSS, Bootstrap, and a small amount of vanilla JavaScript.

## Highlights

- premium hero and listing presentation
- responsive property, service, testimonial, FAQ, and contact sections
- smooth scrolling navigation
- lightweight form validation
- interactive favorite buttons
- local asset setup with no build step

## Getting Started

Open [index.html](/C:/Users/Elbrine/Desktop/shosa/index.html) directly in a browser, or run a simple static server:

```powershell
python -m http.server 8080
```

Then visit `http://localhost:8080`.
