💡 TIPS FOR DEVELOPERS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. CSS Variables
   Use --bg-color, --text-dark, etc. for consistent styling
   Update in :root {} to change site-wide colors

2. Form Validation
   View main.js for form validation logic
   Add new forms by following the same pattern

3. Responsive Design
   All sections use Bootstrap grid (col-md-3, etc.)
   Test with DevTools device emulator

4. Images
   Use ./images/ relative paths
   Supports PNG, JPG, JFIF formats
   Lazy loading implemented automatically

5. Adding New Content
   Follow existing HTML structure
   Maintain consistent class naming
   Update footer links if adding new sections
